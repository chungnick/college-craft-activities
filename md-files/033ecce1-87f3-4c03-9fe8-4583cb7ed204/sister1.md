2026 Advanced Physics Summer Program Application

Welcome to the application form for the UMD Physics 2026 Advanced Physics Summer Program. This program is held at the University of Maryland, College Park. The program is designed for students entering 9th-12th grade this Fall 2026 with an interest in science and math. Please note this is NOT a residential program and there is NO before or after care.  
  
Program Dates: July 20th- 31st, 2026 (2 weeks)

Cost: $375  
      
Please submit any questions to [physicscamps@umd.edu](mailto:physicscamps@umd.edu) with the applicant name, program year, and program name in the subject line.
