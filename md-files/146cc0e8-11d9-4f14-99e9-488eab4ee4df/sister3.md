# M-ASCEND Teacher Summer Program

This program intends to deepen the science-focused skills of teachers in developing cancer-related curriculum materials for high school students, using evidence-based support and hands-on research lab experience. More details will be shared regarding this opportunity.

We invite you to review the information below to learn more about the program timeline and expectations.
