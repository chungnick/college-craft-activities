# Summer Program Tuition Costs

The tuition covers the cost of attending the program. For the on-site summer programs, tuition also includes a shuttle from the airport, housing, and food during the program. Note it does not cover travel costs to and from the program.

| Program | Cost |
| --- | --- |
| Online BootCamp Program | $1,995 USD |
| Online Startup Experience Fall Program | $4,495 USD |
| Online Innovation Summer Program | $4,495 USD |
| Online Flagship Entrepreneurship Summer Program | $6,495 USD |
| In-Person San Diego Exploration Program | $6,495 USD |
| In-Person LocationX Flagship Entrepreneurship Summer Program | $11,495 USD |

An additional $250 international fee for both on-site programs will be charged if you are not from the United States. This fee helps cover international insurance.
