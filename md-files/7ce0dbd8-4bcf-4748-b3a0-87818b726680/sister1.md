# PROGRAM DESCRIPTION

##### I-SITE: I-SITE is an interdisciplinary introduction to key STEM fields drawn from Tandon courses and faculty research. Over five immersive weeks, students engage in coursework, labs, activities, and workshops covering topics such as scientific ethics, public speaking, and presentation training. Students progress through interconnected concepts, using state-of-the-art tools like PyTorch for machine learning. The program culminates in a team challenge where students prototype and present solutions to contemporary societal issues like climate change or cybersecurity, gaining hands-on experience in a dynamic, college-like learning environment.
