# Welcome to Stony Brook's Application Homepage

You will find different types of applications for undergraduate students\* on this homepage and will select an application for one the following: Stony Brook's **[Summer Session](https://www.stonybrook.edu/commcms/summer/index.php)**/**[Winter Session](https://www.stonybrook.edu/commcms/winter/)**, **[FLY program](http://www.stonybrook.edu/commcms/studyabroad/fly/index.php)**.  Login links at bottom.  
  
**\*Do not complete this application if you are a graduate student** (someone who has already completed a bachelor's degree), please **[complete the Non-matriculated Graduate Student Application](https://www.stonybrook.edu/commcms/grad/admissions/apply/non-matriculated-graduate-options.php)**.

#### Winter Session Application (Opens October 13, 2025 )

#### **Instructions:** Please review the [Winter Session Calendar](https://www.stonybrook.edu/commcms/winter/) for registration dates and deadlines. Once the application is open, please allow 3 to 5 business days for processing to receive an email notification with a 9 digit Stony Brook ID number and instructions for registering in SOLAR.

Do not complete this application if you are a:

* **Previous visiting student** (completed a summer or winter course at SBU before), please do not complete this application and send an email to [wintersession@stonybrook.edu](mailto:wintersession@stonybrook.edu)
* **Current Stony Brook student**, please do not complete this application and follow [steps for current students](https://www.stonybrook.edu/commcms/summer/audience/current-students.php)
* **Incoming Stony Brook student** (admitted and coming to SBU for your first semester), please do not complete this application and follow [steps for an incoming student](https://www.stonybrook.edu/commcms/summer/audience/incoming-students.php)

---

#### **SBU Freshman Launch Year (FLY) Program**

This application is not for visiting students. Please select this application only if you are interested in applying for the [Freshman Launch Year (FLY) program](https://www.stonybrook.edu/commcms/studyabroad/fly/index.php). This program is specific for first-year students looking to attend Stony Brook University starting at one of the abroad programs in Korea  for the first year of study. For questions, please contact [flyiap@stonybrook.edu](http://flyiap@stonybrook.edu). There is a $50 application fee for the Spring 2026 FLY Program.
