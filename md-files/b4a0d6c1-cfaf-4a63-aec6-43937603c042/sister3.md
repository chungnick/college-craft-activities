# Sample Past CSIRE Research Projects (Representative Selection)

[Home](/csire) | [Introduction](/csire/introduction) | [Sample Projects](/csire/sampleprojects) | [Past Mentors](/csire/faculty) | [Application](/csire/application) | [News](/csire/news)

# **Biomedical Informatics, Clinical AI & Health Data Science**

* Robust Blood Glucose Management for Insulin Dependent Diabetes Patients with Deep Reinforcement Learning
* Informatics pipeline to retrieve Allele Frequencies and Clinical Significance of DNABERT Candidate Variants Using NCBI dbSNP Database
* Machine Learning-Based Heart Failure Risk Prediction
* Spatial Attention-based Deep Learning System for Breast Cancer PCR Prediction
* A study of Clinical BERTs for Interpretable In-Hospital Mortality Prediction
* Computational Analysis of Somatic Mutation in Cancer Genome
* Predicting Opioid Overdose Using Deep Learning
* Role of Stain Normalization in Computational Pathology: Use Case in Metastatic Tissue Classification
* Single Cell Activity Tracking in Animal Models Using Computer Vision

# **AI, Machine Learning & Data Science**

* Fair Data Distillation for Electronic Health Records Using Neural Networks
* Comparing Results of Models Trained On Different Human Activity Interaction Datasets
* Solving Complex Tasks with Integrated Neural Networks
* AI Research Agent
* Backdoor Attack Against Mortality Prediction with Clinical Notes

# **Computer Vision, Imaging & Graphics**

* Bragg Spot Finder for X-Ray Diffraction Images Using A Machine Learning Approach
* Machine Learning-Driven Characterization of Calcifications in Dental Cone Beam Computed Tomographic Images
* Optimizing Medical CT Image Generation with Style Transfer
* A Computer Vision-based Deep Learning Tool to Assist Residents in Classifying Solid Wastes for Proper Disposal
* cryo-EM Particle Picking
* OCTA Segmentation

# **Human–Computer Interaction, VR/AR & Assistive Technologies**

* Eye Gaze-Based Web Browsing in Virtual Reality
* VR based communication for ALS patients with Meta Quest Pro
* Multi-modal text correction in VR
* Interactive GUI for Real-Time Vital Sign Monitoring and Data Visualization
* Color Enhancement in Keyboards for Visually Impaired Users

# **IoT, Embedded Systems & Systems Research**

* AI Thermostat
* Real-time Strength Training Monitoring, Feedback and Gamification System Using Wearable IMU Sensors
* SMS Phishing
* Lilota (Little Interpreted Language Over the Air)
* Implementing the MQTT Protocol for Use in Home Automation Systems with Lilota

# **Quantum Computing & Mathematical Computing**

* Quantum Kernel Classification for Power System Stability Assessment
* The Effect of Quantum Noise on Entanglement Purification
* Quantum Teleportation
* Finding Optimal Graphs With Given Fixed Numbers of Vertices and Edges
* Proving Alexandrov's Convex Cap Theorem Using Gradients
* Optimal Topology for Minimal Laplacian Energy

# **Security, Privacy & Trustworthy Computing**

* Buffer Overflow Exploits in C (specifically ROP attacks) and mitigation tools
* Improving memory safety of C++ through Rust-C++ interoperability
* Privacy Risks of Smart Home Meters
* Exploring Regular Expressions and their Denial of Service Vulnerabilities

# **Policy, Society & Computational Social Science**

* Using Causal Inference to evaluate the impact of various gun control policies
* Investigating the Factors Affecting Spread of the Opioid Epidemic Using Machine Learning
* Using Social Media to Enhance Rapid Earthquake Response

# **Engineering, Infrastructure & Applied ML**

* Asset-SAM: Segment Anything Model Meets Highway Asset Segmentation and Management
* AI-based Pedestrian Trajectory Prediction and Fairness Evaluation
* A Multi-Modal System to Determine Fatality Information and Infrastructure Damage
* Using Electrical Vehicles to Account for the Instability of the Electrical Grid
