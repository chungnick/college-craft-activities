# How to Apply:

# ​

1. # Review program eligibility requirements here.
2. # Create a CLYLP application account by following the steps outlined [here](https://www.clylp.org/_files/ugd/a994bb_c676e668cec546089f55b52753ba72d8.pdf) (or view the video guide below). \*\*\*DO NOT use school emails\*\*\*
3. # Prepare your information by using the application [checklist](https://www.clylp.org/_files/ugd/a994bb_3c341b144d6a477cae3cd508662f622f.docx?dn=25%20Application%20Checklist.docx).
4. # Begin your online application and add your recommenders information so that they receive their invite.
5. # Submit your application by the deadline.
